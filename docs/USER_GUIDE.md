# ROCm Navigator: User Guide

Welcome to the ROCm Navigator user manual.

## Overview
ROCm Navigator is a multi-agent framework designed to automate the porting of high-performance NVIDIA CUDA kernels to native AMD ROCm codebases.

## Getting Started
1. **Authentication:** Enter your Developer ID and credentials on the login screen to retrieve the active JWT token.
2. **Landing Page:** Click on "Launch Node Editor Workspace" to open the workspace canvas.
3. **Ingest Repository:** Enter your target Git repository path in the search header block.
4. **Execute Porting:** Click the "Test workflow" button. Watch the nodes flash as Scanner, Architecture, Rewrite, Security, Validation, and Performance agents process files.
5. **Diff Analysis:** Inspect code highlights in the comparative code explorer.
6. **Download Reports:** Export comprehensive compilation and occupancy reports.
